How SIP returns are calculated

SIP returns are calculated using compound interest applied monthly. Each monthly investment compounds for a different duration, resulting in long-term wealth creation.

SIP Formula

The SIP maturity amount is calculated using the formula:

M = P × ((1 + r)n − 1) / r × (1 + r)

Where:
P = Monthly investment amount
r = Monthly rate of return (annual rate ÷ 12 ÷ 100)
n = Total number of months

Example

Investing ₹5,000 per month for 10 years at an annual return of 12% demonstrates how disciplined investing can build wealth over time.
