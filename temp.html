How SIP Returns Are Calculated

SIP returns use monthly compounding. Each investment installment earns returns for a different duration, leading to exponential growth over long-term investing.

View SIP Formula

FV = P × [ ((1+r)n − 1) / r ] × (1+r)

P = Monthly investment
r = Monthly rate of return
n = Total number of months
